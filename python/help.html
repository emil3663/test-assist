Test Assist

Annotation & capture tool — Help Guide

Overview

Test Assist is a lightweight always-on-top capture and annotation utility designed for testers and developers. It lets you screenshot or record your screen, annotate the result with shapes, text, and drawings, then copy or save the output instantly.

The interface has two parts: the Floating Launcher (the small overlay panel) and the Editor (the full annotation workspace).

Floating Launcher

Moving the launcher
Drag anywhere on the panel (outside a button) to reposition it freely. Drag it toward the right edge of your screen and it will automatically dock as a compact vertical icon strip. Click the restore arrow icon (← |) to bring back the full panel.
Photo mode Alt+P
Click Quick Capture to draw a selection rectangle over the area you want to capture. Release the mouse to send the screenshot straight to the Editor.

The full-screen icon (monitor button) captures the entire primary display including the taskbar. Keyboard shortcut: Alt+Shift+P.
Video mode Alt+V
Click Start Recording to begin a full-screen recording. A timer shows elapsed time. Click Stop Recording to finish — the file is saved automatically to your home folder.
Dock mode
Click the dock icon (→ | ) in the header row to instantly collapse the launcher to a slim vertical strip pinned to the right edge. The strip shows three icon buttons:
  • TA — opens the Editor
  • Camera — triggers a Quick Capture
  • ← | — restores the full floating panel

Editor

The Editor opens automatically after every capture. It contains a toolbar of annotation tools across the top, the canvas in the centre, and a controls panel on the right.

Annotation Tools

S — Select

Double-click an annotation to select it. Drag to move it. Use the layer buttons in the panel to reorder. Press Delete to remove the selected annotation.

X — Crop

Drag a rectangle to crop the canvas to that region. The original image outside the selection is discarded. Undo (Ctrl+Z) restores the original.

B — Blur

Drag to draw a blur region. Useful for redacting sensitive information. Adjust strength with the Stroke slider.

T — Text

Click anywhere to place a text box. Type your label, then click outside or press Escape to finish. Drag the corner handle to resize the text box.

H — Highlight

Drag to draw a semi-transparent filled rectangle. Adjust fill opacity using the Highlight Fill slider in the panel.

C — Circle

Drag to draw an ellipse. Pick a colour using the colour swatch below the button.

A — Arrow

Drag from the tail to the head of the arrow. Choose Classic, Double, or Dashed style from the Arrow combo box in the panel.

R — Rectangle

Drag to draw an outline rectangle. Great for callout boxes.

P — Pen

Freehand drawing. Hold and drag to sketch paths. Stroke size is controlled by the Stroke slider.

Colour swatches — Each colour-capable tool has a small colour swatch below its button. Click the swatch to open a colour picker and choose a custom colour. Colours are remembered per tool.

Controls Panel (right dock)

Edit section

  • Undo / Redo — step through annotation history.
  • Delete Selected — remove the currently selected annotation.
  • Bring to Front / Send Backward / Send to Back — reorder the layers of the selected annotation.
  • Clear All — remove every annotation (a confirmation dialog appears).

Zoom

Use + / − buttons or the slider to zoom in/out. Click Fit to automatically zoom so the entire image fills the viewport.

Stroke

Controls the line width for most tools (1–20 px).

Arrow style

Switches between Classic, Double, and Dashed arrow styles. Applies to the next arrow drawn and also updates any currently selected arrow.

Highlight Fill

Controls the opacity of the Highlight tool fill (0–100%). Lower values are more transparent; 100% is fully opaque.

Saving & Exporting

💾 Save PNG Ctrl+S
Composites the base image and all annotations into a single PNG and opens a save dialog. The result is also added to the History sidebar automatically.
⧉ Copy
Copies the composited annotated image to the system clipboard so you can paste it directly into Jira, Slack, email, or any other tool.
📋 Export JSON
Saves the raw annotation data as a JSON file. Useful for re-importing annotations or processing them programmatically.

History

Every time you save or copy an image, a thumbnail snapshot is saved automatically to ~/.test-assist/history/. The History sidebar shows your recent snapshots.

Use the Show dropdown to filter: Recent 5, Today, This Week, or This Month. Click the HISTORY header to open a full gallery overlay with all category tabs. Clicking any thumbnail loads that image back into the editor.

Tips & Tricks

  • The launcher stays on top of all windows so you can capture at any time without alt-tabbing away.
  • After capturing, the Editor opens in the background — your flow isn't interrupted. Click the TA button in the launcher (or dock strip) to bring it to the front.
  • Use Crop first to trim unwanted areas, then annotate — the crop is fully undoable.
  • Text tool: after typing, click elsewhere to place the label. Double-click a placed text annotation (with the Select tool) to edit it.
  • Blank/corrupt history snapshots smaller than 5 KB are automatically cleaned up on startup.
